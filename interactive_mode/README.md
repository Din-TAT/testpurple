
Этот скрипт запрашивает у пользователя имя и возраст, а затем выводит персонализированное приветствие.


Скрипт использует команду `read` для получения данных от пользователя во время выполнения:
- Сначала спрашивает имя
- Затем спрашивает возраст
- После этого выводит сообщение вида: `Привет, Имя! Вам Возраст лет.

Как запустить

Сделайте скрипт исполняемым:
chmod +x greet_interactive.sh
Запустите его и следуйте подсказкам на экране
