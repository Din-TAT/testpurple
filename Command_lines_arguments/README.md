Этот Bash-скрипт автоматически создаёт стандартизированную структуру папок для учебных или рабочих заданий
При запуске с названием задания, например `MyTask`, скрипт создаёт:
MyTask/
├── README.md
├── docs/
├── scripts/
└── reports/
Как его использовать?
Сделайте скрипт исполняемым:
 chmod +x setup_task.sh
Запустите, указав название задания как аргумент:
./setup_task.sh "Название_задания"
