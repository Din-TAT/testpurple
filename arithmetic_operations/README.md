Этот скрипт демонстрирует основные способы выполнения арифметических операций в Bash-скриптах.
Используемые методы:
- **`$(( ... ))`** — встроенная арифметика Bash для целых чисел (сложение, вычитание и т.д.)
- **`expr`** — внешняя команда для базовых целочисленных операций (требует экранирования `*`)
- **`bc`** — утилита для вычислений с плавающей запятой

Сделайте скрипт исполняемым:
   chmod +x operations.sh
Запустите его.
